# v0.3.0 Patch Notes

Big rework. Lots of things unmentioned below but these are the highlights:

## New stats
- **Base Spread** – Max angle off target that the weapon can fire when standing still.
- **Move Spread** – Max angle off target while hip-firing and moving normally.
- **Dispersion** – Max distance off target a round can be @200m when standing still (configurable).
- **Bullet Drop** – How far a bullet drops over @200m (configurable).
- **Barrel Wear** – Per-shot chance the barrel itself loses 1% (e.g. `1.2% @ -1%`). Separate from other-part dings; silencers do not change this.
- **Burst Size** – How many ADS shots before recoil climb hits 3 degrees (configurable). Silencers soft recoil can increase this.
- **Hit Power / Dmg vs Stalkers / Dmg vs Mutants** – Clearer dedicated rows (per-ammo where relevant).

## Misc
- Can opt in to having barrel condition / silencer affect the displayed damage numbers.
- Effort was put into making each displayed armor value a REAL value with many NPC that really use it. Some tiers are skipped as only 1-2 models use it, but the damage numbers now map much more accurately to real break points etc. Lots of details in the Info section of MCM config.

## Config
- Large rewrite of the MCM **Info** tab: clearer explanations of what each feature means, how the damage table and enhanced stats are calculated, armor tiers, mutant multipliers, and other Useful Info that was previously thin or missing. Better formatting too!
- Many new MCM options: Highly configurable.
- Added tooltips (fitting...)

## Fixes
Several calculation fixed and more that I don't remember anymore - things *should* be much more accurate.

I have not had a chance to thoroughly test this yet - and it's hard to with so many configurations, so let me know if you have any issues.

## Note on weapon condition
Overall weapon condition is not a real wear metric in G.A.M.M.A. **Guns Have No Condition** keeps it forced around **~83%**. That sits under the WPO barrel-wear threshold (default 85%), so the per-shot Barrel Wear chance stays active. Trust WPO part lines (and Barrel Wear) for durability, not the overall condition %.
